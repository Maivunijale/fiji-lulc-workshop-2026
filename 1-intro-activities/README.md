# Python_Learning_Activities
Activities for teaching Python (estimated time: approx 35-60 minutes)

1. First spend 15-30 minutes on folder 1 activity 1-python-foundations-fiji.ipynb to learn about how to navigate python notebooks within the Jupyter environment
2. Spend 10 minutes on folder 1 activity 2-tc-winston-land-fiji-merge.ipynb
3. The remaining 10-20 minutes can be used to complete folder 2 activity 1-forestry-data-analysis.ipynb

Once complete please let one of the instructors know before proceeding to the next activities.
